Lo Básico

A continuación, se mostrará un código en HTML en su versión 5 en el cual se explicará la estructura básica que lleva un documento HTML versión 5

Etiqueta Atributo Valor Explicacion

doctype html

 X X esta etiqueta hace saber a navegador que va a leer un documento html 5
html lang en Etiqueta html indica inicio del documento, el atributo lang se refiere al idioma del documento y el valor en se refiere al idioma en este caso inglés.
head X X Indica el inicio del encabezado
Meta charset UTF-8 esta etiqueta referencia sobre la página: autor, título, fecha, palabras clave, descripción, etc. atributo charset para codificación de caracteres en el documento HTML Valor UTF-8 valor para la codificación para caracteres Unicode
Meta name-content "viewport"- "width=device-width"

El atributo name se refiere al nombre al que se asocia la meta información, el atributo content se refiere a los datos que se quieren asociar a el atributo name

Los valores viewport es utilizado únicamente por dispositivos móviles y width=device-width establece el ancho de la página para seguir el ancho de pantalla del dispositivo
title X X Esta etiqueta establece el título que va a tener la pestaña del navegador
/head X X Cierre de la etiqueta head.
body X X

Esta etiqueta establece el cuerpo del documento, es decir, lo que se va a visualizar

Nota: html maneja el contenido de manera secuencial de arriba hacia abajo

/body X X Etiqueta de cierre del cuerpo del documento.
/html X X Cierre etiquetas html y final del documento.

Vale aclarar que esta es una explicación es básica, si quieres profundizar más a cerca de más tags y atributos de las mismas puedes ir a paginas como W3school.
