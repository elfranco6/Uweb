¿Qué es HTML?

HTML es una sigla que significa y traduciéndolo al español significa lenguaje de marcas de hipertexto define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos etc. Este leguaje de etiqueta es de tipo secuencial es decir cómo se vaya estructurando así mismo el navegador que es el que se encarga de interpretar ese código lo va a mostrar en pantalla. Este lenguaje comprende de elementos como tags o etiquetas, atributos como src(sources) o recursos id (de identificador) y class (de clases) y muchos más que se explicaran más adelante.

Un poco de historia

HTML hizo su lanzamiento inicial en 1993, HTML es un estándar a cargo del World Wide Web Consortium que es organización dedicada a la estandarización de casi todas las tecnologías ligadas a la web a lo largo del tiempo este leguaje de marcado ha sufrido cambios para que fuese más eficiente facilitar el desarrollo de páginas web compatibles con distintos navegadores y plataformas. La actual versión de este lenguaje de etiquetado es la versión 5 que fue lanzada el 28 de octubre de 2014

Nota: HTML NO es un lenguaje de programación como lo es JAVA, .NET o PHP entre otros.
