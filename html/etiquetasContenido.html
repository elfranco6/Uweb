Etiquestas de contenido

Las Tags Contenido son aquellas que albergar cierto contenido

estas etiquetas ayudan a que un documento tenga semántica

etiquetas Atributos Valores Explicacción
Heading

Class, id

Este elemento no tiene atributos que no sean los atributos globales, comunes a todos los elementos.

También conocidas como las etiquetas estas significan encabezado

van desde h1 hasta h6 según su importancia así h1 sería la más importante y h6 la menos importante.

p

Class, id

Este elemento no tiene atributos que no sean los atributos globales, comunes a todos los elementos.

El texto que esté dentro de la etiqueta párrafo el navegador entenderá que el mismo es un párrafo
Article

Class, id

Este elemento no tiene atributos que no sean los atributos globales, comunes a todos los elementos.

Esta etiqueta define un contenido específico dentro de una web y que por sí solo tiene sentido.
section

Class, id

Este elemento no tiene atributos que no sean los atributos globales, comunes a todos los elementos.

Esta etiqueta define sirve para definir una sección o agrupación temática del contenido de una página web.
Aside

Class, id

Este elemento no tiene atributos que no sean los atributos globales, comunes a todos los elementos.

Esta etiqueta define una sección de la web donde el contenido no tiene que estar necesariamente relacionado con el contenido principal de la web.

Generalmente suelen ser barras laterales o sidebars de una web (Plugins sociales, publicidades…).
Footer

Class, id

Este elemento no tiene atributos que no sean los atributos globales, comunes a todos los elementos.

Contiene la agrupación de código HTML correspondiente al pie de página que suelen tener

habitualmente todas las webs como redes sociales datos de contacto y copyright
