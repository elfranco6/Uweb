Buenas practicas

En esta sección se hablará de buenas prácticas que se deben tener

en cuenta para codificar no se convierta en un infierno.

  1. Utilizar las etiquetas especificas del lenguaje y no mucho la etiqueta div ya que ayuda a que el documento tenga más semántica.

  2. Enlazar los archivos JavaScript a lo último del documento ya que con esto reduciremos mucho el tiempo de carga de la página web

    recuerda que html carga todo de manera secuencial.

  3. Estar pendiente del cierre de las etiquetas ya que, si no se cierran el resultado no será el que se espera.

  4. Usar plugin o herramientas que permitan definir los elementos básicos de un documento html.
