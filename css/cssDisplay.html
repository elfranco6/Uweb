Display

Display es una propiedad de css que especifica el tipo de cuadro utilizado para un elemento HTML estas pueden tener variaciones como:

Propiedad Valor Explicación
display Inline Valor por defecto. Muestra un elemento como un elemento en línea.
display block Muestra un elemento como un elemento de bloque.
display inline-block Muestra un elemento como un contenedor de bloques en línea. El interior un bloque
