Existen más comando de git estos son:

Comando Git Explicación
git init Este comando se usa para crear un nuevo repertorio GIT
git add Este comando puede ser usado para agregar archivos al repositorio
git clone Este comando se usa con el propósito de revisar repertorios. Si el repertorio está en un servidor remoto
git commit El comando commit es usado para cambiar a la cabecera
git status Este comando muestra la lista de los archivos que se han cambiado junto con los archivos que están por ser añadidos
git push Este es uno de los comandos más básicos. Un simple push envía los cambios que se han hecho en la rama principal de los repertorios remotos que están asociados con el directorio que está trabajando
git checkout se puede usar para crear ramas o cambiar entre ellas
git remote se usa para conectar a un repositorio remoto
git branch Este comando se usa para listar, crear o borrar ramas.
git pull Se utiliza para poder fusionar todos los cambios que se han hecho en el repositorio local trabajando
