Instalación Github

Instalar Github en tu computadora es muy fácil.

Sigue este paso a paso y veras lo sencillo que es instalar github en tu computadora.

  1. primero ir a sitio web de git https://git-scm.com para descargar el archivo ejecutable.

  2. segundo luego de haber descargado el archivo ejecutable de va a la carpeta contenedora, se selecciona y se da clic derecho y ejecutar como administrador.

  3. tercero dar clic en el botón next hasta que el botón next cambie a install.

    Nota: se recomienda no cambiar ni palomear ninguna opción diferente a las que están por defecto en la instalación.

  4. cuarto luego de que el botón esté en install damos clic en install y esperamos a que termine el proceso de instalación.

  5. quinto finalmente damos clic en el botón para cerrar el asistente de instalación.

    6. Nota: este paso a paso está basado para sistema operativo Windows 10 pero, puede servir a versiones anteriores del mismo.
